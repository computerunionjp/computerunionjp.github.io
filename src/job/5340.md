#### 必要スキル

下記必須スキルより3つ以上満たしている方

- AWS・GCP・Azureのいずれかの設計～運用のご経験
- Docker及びその周辺技術(ECS, Kubernetes, Istioなど)を用いた設計～運用のご経験
- Infrastructure as Code（Terraform, CloudFormation, CDKなど）を実践されたご経験
- CI/CDツール（CircleCI, GitHub Actions, CodeBuild, CloudBuild, Jenkins等）を利用したご経験
- さまざまなデータストア（MySQL, PostgreSQL, BigQuery, Elasticsearch, S3, GCSなど）の特性を理解し、設計～運用されたご経験

- Terraformを利用したインフラ構成変更およびそのレビュー
