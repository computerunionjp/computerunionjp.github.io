#### 必要スキル

※最低限としてDWHとAzure SQL Databaseの経験あれば検討対象

- DWHプロジェクト開発またはリプレース経験者
- 20TB以上、1テーブル当たり数億規模のデータベース経験者
- Azure SQL Database経験者
- 設計～本番移行までのプロジェクト経験

#### 業務内容

現在のDWH（オンプレ）をAzureクラウド環境にリプレースする。Azure SQL Databaseに関して、設計、開発から外部結合テスト、総合テスト、データ移行/本番移行までを担当。

- クライアントのPL/SQLメンバに対して、T-SQLとの違いやテストの観点などのリーディングが主作業
- 加えてテスト計画やデータ移行計画の検討等の技術支援を行う
