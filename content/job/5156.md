#### 業務内容

証券会社のAzure環境を中心とした環境構築・保守業務を行っております。一部、タブレット端末利用者からの問合せ対応等もございます。顧客のシステム子会社のチームに入り、内製での構築支援を中心に実施となります。

- ポジション1: Azure全般の運用（Azure環境全般の問合せ、調査、設定変更、構築等）
- ポジション2: Windows Server オンプレ基盤構築やテスト・検証の支援

#### 必要スキル

- ポジション1
  - Azure Virtual Desktop環境の構築、運用経験
  - Azure構築経験
  - Microsoft Intuneのご経験
- ポジション2
  - サーバー設計構築経験
  - 運用ツール作成経験（SQL、VBA）
  - 運用管理環境設計・構築（JP1）
